Pour installer l'application, sur iPhone ou iPad, ajoutez la à l'écran d'accueil en appuyant sur l'icône de partage, puis sélectionner "Sur l'écran d'accueil".
Pour installer cette application sur votre ordinateur Mac , suivez les étapes ci-dessous :
  1. Ouvrez Safari
  2. Cliquez sur "Fichier" dans la barre de menu
  3. Sélectionnez "Ajouter à l'écran d'accueil"
