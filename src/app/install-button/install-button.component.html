Pour installer cette application sur votre ordinateur Mac , suivez les étapes ci-dessous :
  1. Ouvrez Safari
  2. Cliquez sur "Fichier" dans la barre de menu
  3. Sélectionnez "Ajouter à l'écran d'accueil"
Vous pouvez installer cette application en l'ajoutant à l'écran d'accueil de votre iPhone ou iPad. Appuyez sur l'icône de partage, puis sélectionnez "Ajouter à l'écran d'accueil".
